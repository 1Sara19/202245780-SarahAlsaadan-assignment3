# Assignment 2 – Interactive Features

This project is an improved version of my personal portfolio website from Assignment 1.  
It focuses on adding interactive features, dynamic behavior, and better user experience.

## Features
- Responsive design for desktop, tablet, and mobile
- Light/Dark mode toggle with saved preference using localStorage
- Dynamic greeting message based on time of day
- Project filtering by category (Web / Database)
- Contact form with validation and success/error messages
- Smooth scrolling and UI transitions
- Interactive UI elements (hover effects, animations)

## Dynamic Features
- Greeting notification changes based on the current time
- Projects section updates dynamically when filtering
- Form feedback updates based on user input

## Data Handling
- Theme preference stored using localStorage
- Form validation with user feedback messages

## How to Run Locally
1. Clone or download the repository
2. Open the project folder
3. Open `index.html` in your browser
